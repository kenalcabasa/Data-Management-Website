# Data Management Website
This simple project is a Data Management Website where it features the datasets from NYC open data. 

This website is a simple data management platform that fetches real-time public datasets from NYC Open Data. It allows users to view, organize, search, and manage city-related information through an easy-to-use interface. The system is designed to make open government data more accessible and understandable by presenting it in a clean and structured format.

Features include:

Real-time data fetching from NYC Open Data APIs
Search and filtering functionality
Organized table and data views
Simple and responsive user interface
Efficient data management and visualization

The project demonstrates the use of APIs, data handling, and web development technologies to create a practical and user-friendly data management solution.
